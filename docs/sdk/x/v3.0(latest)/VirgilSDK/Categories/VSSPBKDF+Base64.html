VSSPBKDF(Base64) Category Reference

Declared in VSSPBKDF+Base64.h

Overview

Helper category to compose Base64 encoded password-based derived key.

– base64KeyFromPassword:size:error:

Composes a key using PBKDF from the given password. Resulting key (before base64) will have required size.

- (NSString *__nullable)base64KeyFromPassword:(NSString *__nonnull)password size:(size_t)size error:(NSError *__nullable *__nullable)error

Parameters

password

Stirng password which is used to generate a key.
size

Required size of the key.
error

NSError pointer to return an error.

Return Value

Base64 encoded key data in case of success. If error happened - returns nil and NSError object as a last parameter.
